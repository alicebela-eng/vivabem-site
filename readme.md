Landing Page - VivaBem Medicina Integrada
Este projeto consiste em uma Landing Page Estática desenvolvida para a Clínica VivaBem, focada em modernizar a experiência tecnológica dos pacientes através de uma apresentação clara de serviços e equipe
.
Como executar o projeto
Como este é um projeto de página estática (HTML e CSS), não é necessária a instalação de dependências ou servidores complexos. Para visualizar a página, siga os passos abaixo:
Clone o repositório ou baixe os arquivos:
Clique no botão verde Code e selecione Download ZIP ou utilize o comando:
Acesse a pasta do projeto:
Navegue até o diretório onde os arquivos foram salvos.
Abra o projeto no navegador:
Localize o arquivo index.html e dê um clique duplo nele.
O projeto será aberto automaticamente no seu navegador padrão (Chrome, Firefox, Edge, etc.).
Dica: Se você ativou o GitHub Pages, a página também pode ser visualizada online através do link: [SEU LINK DO GITHUB PAGES AQUI]
.
Tecnologias Utilizadas
HTML5: Estruturação de conteúdo e formulários
.
CSS3: Layout responsivo e estilização visual.
Requisitos Implementados (Checklist)
De acordo com as especificações do projeto, esta landing page contém:
Título da aba customizado
.
Mínimo de 2 imagens de serviços (Cuidado integrado e Atendimento médico)
.
Descrição detalhada dos serviços e exames laboratoriais/imagem
.
Equipe fictícia com 3 membros (Dr. Pedro, Dra. Marisa e Dra. Marina) incluindo fotos e cargos
.
2 níveis de cabeçalhos (H1 e H2/H3) para melhor hierarquia
.
Formulário estático capturando Nome, E-mail, Cidade e Estado
.
